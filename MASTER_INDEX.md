## üí° Pro Tips

### Tip 1: Use npm Scripts
Don't memorize commands. Use:
```bash
npm run        # Shows all scripts
npm run test   # Run tests
npm run health # Check API
```

### Tip 2: Read Documentation in Order
Start with the QUICK reference, then deep dives as needed.

### Tip 3: Keep Terminals Open
- Terminal 1: Backend (`npm run server`)
- Terminal 2: Frontend (`npm run dev`)
- Terminal 3: Commands/tests

### Tip 4: Use jq for JSON
```bash
curl http://localhost:5000/api/banks | jq .
# Much prettier than raw JSON!
```

### Tip 5: Test Before Deploying
Always run `npm run test` before pushing to production.
